# Running on Ubuntu  

To run the project on Ubuntu, follow these steps:  

1. Place the `minilibx-linux` folder inside the `lib` directory.  
2. Use the following key mappings in your code:  

## Key Mappings  

```c
#define KEY_A 97
#define KEY_S 115
#define KEY_D 100
#define KEY_W 119
